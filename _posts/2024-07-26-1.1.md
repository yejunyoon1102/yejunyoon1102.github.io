# 1.1 인공지능과 머신러닝, 딥러닝

### 인공지능

사람처럼 학습하고 추론할 수 있는 지능을 가진 컴퓨터 시스템을 만드는 기술

강인공지능 : 사람과 구분하기 어려운 지능을 지닌 컴퓨터 시스템(ex. 자비스)
약인공지능 : 특정분야에서 사람의 일을 도와주는 보조역할

### 머신러닝

규칙을 일일이 프로그래밍하지 않아도 자동으로 데이터에서 규칙을 학습하는 알고리즘을 연구하는 분야
통계학과 깊은 관련!!

대표적 라이브러리 : **사이킷런(scikit-learn)**
파이썬 API를 사용하기 때문에 대중적이고 편리함

### 딥러닝

머신러닝 알고리즘 중에 인공신경망을 기반으로 한 방법들을 통칭하여 딥러닝이라고 함!!

대표적 라이브러리 : **텐서플로(TensorFlow)**
